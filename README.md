# amiibo Card Manager

あつまれ どうぶつの森の amiiboカードを、カード番号中心で管理する単体HTMLアプリです。

## Features

- 001-448のカード一覧
- カード番号・キャラクター名検索
- 所持、未所持、複数所持の管理
- ダブり枚数の入力
- 所持数、コンプ率、ダブり数の表示
- シリーズ、SPカード、所持状態での絞り込み
- CSVエクスポート、インポート
- ブラウザ内保存
- プリセットテーマ、カスタムテーマ

## Usage

`amiibo-card-manager.html` をブラウザで開くだけで使えます。

## Data Source

カード名は任天堂公式サイト「どうぶつの森 amiibo一覧」の第1弾から第5弾の掲載情報を参照しています。
